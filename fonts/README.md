# ObscuraVT Stamp Fonts

Custom display / stamp fonts for the classified, top-secret, government-agent aesthetic in ObscuraVT (VTuber discovery hub).

## Fonts Added

- GovtAgentBB (regular + italic variants in .ttf/.otf)
- ArchiveStamp (.otf)
- Top Secret Stamp (.ttf)

Binary font files are available in the project attachments for integration via next/font/local.
